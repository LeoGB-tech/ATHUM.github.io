Guide d'installation & Sécurité

1

Après le téléchargement, lancez ATHUM.exe.

2

Dans la fenêtre Windows Defender, cliquez sur "Informations complémentaires".

3

Cliquez sur "Exécuter quand même" pour finaliser l'installation.

Note : Si le téléchargement ne se lance pas, essayez d'ouvrir le lien dans un nouvel onglet.
