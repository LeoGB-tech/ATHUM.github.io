⚠️ Windows blocks the installation?

Because Athum is brand new, Windows SmartScreen doesn't recognize it yet and might block the installer. This will improve over time as more people install it. Until then, you have two options:

Quick fix: On the SmartScreen warning, click "More info", then "Run anyway".

  1. Open Windows Security (search "Windows Security" in the Start menu)
  2. Click App & browser control
  3. Click Reputation-based protection settings
  4. Turn Off the option Check apps and files
  5. Install Athum normally
  6. Re-enable it after installation by switching it back On — recommended for your security

💡 Athum is safe — it's an open-source Electron app. The SmartScreen warning only happens because the developer (a 13-year-old) doesn't have a $199/year Microsoft code-signing certificate yet. This is normal for new indie apps.
